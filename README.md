### About

I'm a **backend, AI, and full-stack engineer**. Most days I'm writing production services in Java and Spring Boot, building AI features in Python, or shipping frontend in TypeScript. The interesting problems for me are where these three meet — a backend that scales, an AI feature that survives in production, a distributed system that doesn't fall over at 3am.

I care more about fundamentals than frameworks. System design, distributed systems, clean architecture — these are the things that compound. I'd rather understand something properly than pretend it's a solved problem.

Beyond that, I keep up with problem solving on LeetCode (~134 problems so far) and I'm planning to ramp the volume up significantly.
